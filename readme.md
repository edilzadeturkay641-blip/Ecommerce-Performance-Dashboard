# Ecommerce Performance Dashboard

Interactive dashboard built with Streamlit, PostgreSQL and Plotly.

## Features

- Sales analysis
- Customer segmentation
- Top products
- Top customers
- Sales by category
- Sales by city

## Technologies

- Python
- Streamlit
- PostgreSQL
- SQLAlchemy
- Pandas
- Plotly

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
